# Contributing

Thanks for improving Dutch SME Acquisition Scout.

1. Create a focused branch from `main`.
2. Install dependencies with `pnpm install`.
3. Keep API keys in `.env.local`; never commit them.
4. Add or update tests for behavior changes.
5. Run `pnpm check` before opening a pull request.

Please keep acquisition claims evidence-backed. Unsupported financial, employee, ownership, and succession information must remain explicitly unavailable.
